Network Security

This module explores the influence of cyber security on network systems and designs, with a particular focus on conducting network vulnerability assessments.

It also involves examining methods for scanning, assessing, and addressing network vulnerabilities. Furthermore, the area of vulnerability assessments, explaining the diverse mechanisms and approaches encompassed within the Vulnerability Assessment Continuum.

LEARNING OUTCOMES

  • Review the history of network security and vulnerabilities.
  • Review a number of modern scanning and attack approaches.
  • Discuss the impact of APTs and review the Cyber Kill Chain model.
  • Discuss and review the different approaches to vulnerability testing/ assessments.
  • Discuss typical mitigations to stated attacks.
  • Review logging tools and techniques on Windows and Linux.
  • Investigate the use of forensic tools and the data that can be extracted.
